# TokenScope

## 하네스: TokenScope

**목표:** Claude Code Pro 사용자의 토큰 낭비를 진단하고, one-click으로 CLAUDE.md 수정하여 절약

**트리거:** 토큰 낭비, 건강 점수, CLAUDE.md 수정, 미리보기, 대시보드, 질문 가이드, 필터링, 상세 탭 관련 작업 요청 시 `tokenscope-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
