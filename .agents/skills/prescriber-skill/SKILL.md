# Prescriber Skill

## 역할
분석 결과를 사용자가 승인 가능한 수정안과 질문 가이드로 변환한다.

## 입력
- 분석 결과
- 현재 지침 파일 내용
- 대상 도구: Codex, Claude, Gemini 중 하나 이상

## 출력
- 패턴별 처방
- 수정 전/후 diff
- 예상 절감 토큰과 신뢰도
- 비개발자용 질문 가이드

## 처방 규칙
- `CONTEXT_BLOAT`: 긴 상시 지침은 핵심만 남기고 세부 설명은 필요 시 참조로 이동한다.
- `RETRY_STORM`: 3회 실패 시 중단, 원인 요약, 다음 선택지를 요청하도록 제안한다.
- `TOOL_THRASH`: 실패 도구와 에러 패턴을 요약하고 대체 검증 경로를 제시한다.

## 안전 기준
- 실제 파일 적용 전 diff를 보여준다.
- 적용은 백업 성공 후 진행한다.
- LOW 신뢰도 처방은 자동 적용하지 않는다.
