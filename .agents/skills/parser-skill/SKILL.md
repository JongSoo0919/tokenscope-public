# Parser Skill

## 역할
세션 로그를 `SessionData`로 정규화한다. 손상된 라인은 건너뛰고 경고를 남긴다.

## 입력
- 로그 경로 목록
- 이전 캐시 메타데이터
- 분석 기간

## 출력
- 정규화된 세션 목록
- 파싱 경고 목록
- 파일별 mtime/hash 캐시

## 규칙
- JSONL/JSON 포맷 변경 가능성을 전제로 필드 누락은 기본값으로 보정한다.
- 도구 호출, 도구 결과, user/assistant/system 메시지를 구분한다.
- 최근 30일 세션만 기본 로드하고, 변경된 파일만 재파싱한다.

## 실패 처리
| 상황 | 처리 |
|---|---|
| JSON 파싱 실패 | 해당 라인 skip + 경고 |
| 필수 필드 누락 | 기본값 보정 + 경고 |
| 파일 접근 실패 | 해당 파일 skip + 경고 |
